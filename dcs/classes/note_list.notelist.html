Class NoteList

Clase que permitirá manejar las notas de los usuarios para los diferentes comandos a implementar. Para esta practica se ha modificado el contenido de la clase haciendo que en los diferentes metodos que incluye se haga uso de la API de node.js para la manipulación de ficheros en vez de realizar esta tarea en el archivo note-app.ts de la práctica 8.
